# A Portfolio Website

## Description
**Portfolio website**
**The website is entirely responsive, mobile first**
- Production version v0.1
- Portfolio website, showing different features and holding links to the projects I did or participated in

## Functionality & Design elements utilized
- Semantic HTML elements
- CSS Flexbox
- AVIF image format
- Animated Hamburger menu
- Intersection Observer for animated cards
- Lazy loading of images
- Reusable components
- Email service connected - Resend
- Contact form validation
- Dark Mode - nav menu icon
- React Router with custom 404 page

https://reactrouter.com/ -->

- Cookies implementation - CONSENT POPUP
- Privacy Policy and Terms of Use subpages
